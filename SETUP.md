# DE Dental Square — Merged Premium Website

## What's Included
- ✅ Repo 1: Beautiful UI design (glassmorphism, blobs, animations)
- ✅ Repo 2: Full functionality (Google Auth, appointments, admin dashboard, Firebase)
- ✅ Repo 3: Spline 3D anti-gravity scene in hero
- ✅ New: Premium animations, scroll reveals, 3D card tilts, custom cursor, marquee

## Setup

```bash
npm install
```

Create `.env` (copy from `.env.example`):
```
VITE_FIREBASE_API_KEY=...
VITE_FIREBASE_AUTH_DOMAIN=...
VITE_FIREBASE_PROJECT_ID=...
VITE_FIREBASE_STORAGE_BUCKET=...
VITE_FIREBASE_MESSAGING_SENDER_ID=...
VITE_FIREBASE_APP_ID=...
```

```bash
npm run dev
```
